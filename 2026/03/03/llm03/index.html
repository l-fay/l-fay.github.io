大模型时代的一些思考

发表于 2026-03-03 分类于 LLM 阅读次数： 本文字数： 869 阅读时长 ≈ 3 分钟

好久没有写博客的冲动了，强迫自己记一下最近的一些想法。

ChatGPT

最初ChatGPT横空出世的时候，作为一个在深度学习领域水过论文的人，大概了解了一下Transformer和涌现，并没有给我的心里带来多大的起伏。只是觉得，“哦，NBNB”。

当时也就是利用ChatGPT来写一些毫无营养的公文，并没有更深入的应用。

DeepSeek

然后就是DeepSeek。

从ChatGPT时代开始，就一直有人在研究思维链之类的东西，所以深度思考也并不让我很意外。

只是从DeepSeek开始，国内各家搭载DeepSeek模型的平台可以让我很方便地用上，感觉方便了许多。

DeepSeek对我最直接的的冲击，就是促使我买了张大显存的显卡自己本地化部署模型玩了玩，淘汰了我旧的垃圾显卡，阴差阳错地躲过了硬件大涨价。

所以我很感谢DeepSeek。

在DeepSeek时代中，我还自己串了个工作流，一直用到现在，本质上还是个玩具。

MCP

这一波AI浪潮里，最震撼我的莫过于MCP的出世。

MCP的出现，意味着AI影响现实世界的方式更加直接，可以做到更多的事情。

从MCP面世以来，AI的应用技术简直一天一个样子，太疯狂了。

OpenClaw和具身智能

然后很快，就是OpenClaw。

说实话，我觉得很多人吹OpenClaw是为了卖课或者卖搭载了OpenClaw的Mac Mini给不懂的人，实际上大多数人并没有使用OpenClaw的场景。

但这并不代表OpenClaw就不行。

我认为OpenClaw是一个里程碑，它代表着目前AI应用技术的巅峰。大多数人用不到只是因为它在现实的物理世界没有配套的手脚，并且它还没有摧毁某个具体的行业。

具身智能也在发展，Agent作为大脑，具身智能作为身体，这不就是一个人吗？人能做的，它终究都能做的。

结尾

回到开头，好久没有写博客的冲动了，因为大多数前AI时代的疑难杂症可以在几轮对话里轻松解决。解决地太轻松，都没感觉到有什么坑，所以自然也没什么想记下来的东西。对比这两篇文章：第一篇、第二篇，这个感觉越发明显。

最近一段时间，AI发展恨不得一年走完人类五百年的路。

贾维斯已经来了，奥创也许正在路上。我担心吗？我担心个P，天塌了有个子高的顶着。

我只是感到有些焦虑，我仿佛有些理解了，改革开放时期，人们面对时代浪潮的不同想法。

一些人抓住新机遇乘风起飞；一些人面对时代剧变无可适从；还有一些人，甚至没有意识到世道已经变了，只能在几十年后后知后觉地说，“如果我当年……”。

我现在就是第二种人，不知道在这个世道里，能利用AI做什么。但至少我还在探索，先积累一些知识和经验总不会错。我希望当有一天，我的积累足够的时候，能够开开智，让自己的生活再上一个台阶吧。
